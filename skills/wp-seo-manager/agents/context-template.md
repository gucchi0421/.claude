# エージェント間引継ぎ書テンプレート

各フェーズ完了時にこのテンプレートを埋めてディレクターに渡す。
ディレクターが次のエージェントに渡す際は不要な項目を削除してよい。

---

## 基本情報

```
実施日: YYYY/MM/DD
対象サイト: https://your-site.com
現フェーズ: audit / research / plan / execute / verify（いずれか）
次のエージェント: ディレクター / SEOエンジニア / ライター / コーダー
```

---

## ディレクター → ライター（承認・バックアップ完了後）

```
対象ページ: [URL]
対策KW: [KW]
方向性: [SEOエンジニアのメモをそのまま]
追記してほしい内容: [競合ギャップの要素]
目標文字数: 現在XX文字 → +XXX文字（合計XXX文字程度）
FAQを X件生成してください

```

## ディレクター → コーダー（承認・バックアップ完了後）

