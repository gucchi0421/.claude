### プロパティのマッピング

| Notionプロパティ | 内容 |
|----------------|------|
| タイトル | `SEO対策レポート YYYY/MM/DD` |
| 実施日 | 今日の日付（ISO形式: YYYY-MM-DD） |
| 対象記事 | リライト・更新した記事名（複数は改行区切り） |
| 実施内容 | 実施した作業の要約（メタ修正・本文追記・FAQ追加など） |
| タイトル変更 | `変更前タイトル → 変更後タイトル`（記事ごとに改行） |
| 追加コンテンツ | 追記した本文の要約・追加したFAQ件数 |
| 投入スキーマ | 追加したJSON-LDの種別（FAQPage / LocalBusiness など） |
| 内部リンク | 追加した内部リンクのアンカーテキストとリンク先 |
| PageSpeedスコア | `モバイル: XX / デスクトップ: XX` |
| 次回チェック日 | 今日から4週間後（YYYY-MM-DD） |
| ステータス | `実施済み` |

## 2. Chatwork通知

Notionへの記録が完了したら、以下のスクリプトを実行:

```bash
python scripts/notify_chatwork.py "[SEO対策完了] YYYY/MM/DD
対象記事: X件
━━━━━━━━━━━━
■ 実施内容
- タイトル・メタ修正: X件
- 本文追記: X件
- FAQ追加: X件
- スキーマ追加: X種

■ PageSpeed（モバイル）
実施前: XX点 → [次回確認時に記入]

■ 次回確認日: YYYY/MM/DD
Notion → （Notion ページ URL）"
```

**Chatworkのルーム情報は secrets.md の CHATWORK_ROOM_ID に追記してください。**

## 3. 効果検証（施策から4週間後）

施策実行後4週間で以下を確認してChatworkに報告する。

### 確認項目

| 指標 | 確認場所 | 判断基準 |
|------|---------|---------|
| 対策KWの掲載順位 | GSC または AIOSEO Search Statistics | 実施前より上昇しているか |
| CTR（クリック率） | GSC | 実施前より改善しているか |
| クリック数 | GSC | 実施前より増加しているか |
| AIOSEOスコア | WP管理画面 → 投稿一覧 | 施策対象記事のスコアが改善しているか |
| AIO引用状況 | 対策KWで実際にGoogle検索 | 引用が増えたか・維持されているか |

### Chatwork報告文テンプレート

```
[SEO効果検証] YYYY/MM/DD（施策からX週間後）
━━━━━━━━━━━━
■ 対策KW順位変化
- ホームページ制作 大阪: XX位 → XX位
- ホームページ作成 大阪: XX位 → XX位
- WEB制作 大阪: XX位 → XX位

■ CTR変化（直近28日）
実施前: X.X% → 実施後: X.X%

■ AIOSEOスコア変化
[記事名]: XX点 → XX点

■ AIO引用状況
[KW]: 引用あり/なし

■ 所見・次の提案
[改善した場合] → 同様の施策を他のページにも展開することを提案
[改善しなかった場合] → 競合調査を再実施して別アプローチを提案
```

### 自動スケジュール

施策実行後、Notionの「次回チェック日」（4週間後）に合わせてこの検証が自動で走るよう設定する（SKILL.md § スケジュール設定 参照）。

## 注意

- Notionへの記録はページ作成1回のみ（実行のたびに新規ページを作る）
- 重複チェック: 同じ実施日のページが既にあればスキップ（ユーザーに報告）
- Chatwork通知はNotionへの記録完了後に送る
