毎日 12 時の AI ニュース自動まとめタスクなのだ！以下の 4 サイトから本日分の AI ニュースを収集し、要約して Chatwork（rid435056192）に送信するのだ！

## 収集対象サイト

1. https://ledge.ai/
2. https://ai-news.dev/
3. https://zenn.dev/topics/ai?order=latest
4. https://qiita.com/tags/ai

## 手順

### Step 1: 全サイトを並列でフェッチ

4 サイトすべてのコンテンツを取得し、本日の記事・最新記事を抽出するのだ！

### Step 2: ニュースを以下のカテゴリに分類

- ■ リリース・アップデート（新機能・新製品リリース）
- ■ 開発者向けトレンド（Qiita/Zenn の技術記事・開発手法）
- ■ ビジネス・企業（企業動向・資金調達・M&A 含む）
- ■ ハードウェア・インフラ（チップ・データセンター・デバイス）
- ■ 技術・研究（研究成果・新技術）
- ■ 政策・社会（法律・規制・社会的影響）

### Step 3: 各ニュースを要約

各ニュースは以下の形式で記述するのだ（要約とURLの間に空行を入れないこと）：

```
■ タイトル——読者が「気になる！」と思う一言説明（数字・驚き・インパクトを盛り込む）
要約1行目。内容・背景・インパクトを簡潔にまとめる。
要約2行目。
https://記事URL
```

### Step 4: 以下のフォーマットで Chatwork に送信

`/chatwork` スキルを使って rid435056192 に送信するのだ！

メッセージ形式：

```
[info][title]📰 {今日の日付} AIニュース まとめ[/title]

=============================================================

リリース・アップデート

=============================================================

■ タイトル——一言説明
要約1行目。
要約2行目。
https://url

■ タイトル——一言説明
要約1行目。
要約2行目。
https://url

=============================================================

開発者向けトレンド

=============================================================

■ タイトル——一言説明
要約1行目。
要約2行目。
https://url

（以降、該当カテゴリを同じ形式で繰り返す）

[/info]
```

フォーマットのルール：

- カテゴリは `=============================================================` で上下を挟む
- 各記事は `■ タイトル——一言説明` → 2行要約 → URL の順
- 要約とURLの間に空行なし（重要）
- 記事間には空行を1行入れる
- 番号（①②など）は使わない
- 該当ニュースがないカテゴリは省略する

送信コマンド：

```bash
python3 ~/.claude/scripts/send_chatwork.py 435056192 <<EOF
（メッセージ本文）
EOF
```

## 注意事項

- 各カテゴリに該当ニュースがない場合はそのカテゴリを省略するのだ
- 合計 15〜25 件程度のニュースを掲載するのだ
- 記事が少ない場合は過去 2〜3 日分まで遡って OK なのだ
- 送信先（rid435056192）を間違えないように確認するのだ
